Fes un programa **exercici007.py** que faci el seg√ºent dibuix, a partir de dos bucles:

- Els bucles es defineixen com: **"for q in range (0, len(colors))"**

- Els colors del primer bucle s'obtenen de la seg√ºent llista:
    **"colors = [(127, 184, 68), (240, 187, 64), (226, 137, 50), (202, 73, 65), (135, 65, 152), (75, 154, 217)]"**

- Els colors *grisos* s'obtenen din√†micament des de (0, 0, 0) sumant 25 a cada iteraci√≥. √âs a dir: **"(0, 0, 0) > (25, 25, 25) > (50, 50, 50) ..."**

- La mida dels quadres √©s de **50** i els radis de la resta de pol√≠gons √©s de **25**

- Necessitar√†s la seg√ºent funci√≥ per dibuixar els triangles i pent√†gons.

```python
def draw_polygon(screen, color, center, radius, num_vertices, angle_offset=(math.pi / 3)):
    points = [
        (
            center[0] + radius * math.cos(angle_offset + i * 2 * math.pi / num_vertices),
            center[1] + radius * math.sin(angle_offset + i * 2 * math.pi / num_vertices)
        )
        for i in range(num_vertices)
    ]
    pygame.draw.polygon(screen, color, points)
```

def app_run():
    global time
